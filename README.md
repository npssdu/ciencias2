## Busquedas Internas

### Funcionalidades

La aplicación de Búsquedas Internas permite utilizar los siguientes algoritmos y funciones:

- **Algoritmos de Búsqueda:**
  - **Búsqueda Lineal**
  - **Búsqueda Binaria**

- **Funciones Hash:**
  - **Hash Mod**
  - **Hash Cuadrado**
  - **Hash Plegamiento**
  - **Hash Truncamiento**
